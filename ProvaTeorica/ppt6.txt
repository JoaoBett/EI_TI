###Malware###

Virus -> Código executavel malicioso que pode estar incluido num programa legitimo
      -> Precisa da ação do utilizador para ativar e propagar


Worms -> Código malicioso que se multiplica explorando vulnerabilidades
      -> Executa-se a si próprio

Trojan Horse -> Carrega código malicioso mas disfarçado
             -> Precisa de ações do utilizador

Logic Bomb -> Programa malicioso mas adormecido
           -> "Acorda" com a execução de um programa ou um horario especifico

Ramsonware -> Pede resgate monetário para libertar os dados
           -> Cifragem de dados com chave desconhecida

Backdoors -> Permite acessos remotos de utilizadores não autorizados
          -> Permite o criminoso voltar

Rootkit -> Altera ficheiros para se esconder

Spam -> método de publicidade

Spyware -> recolher informaçoes

Adware -> massacrar o utilizador com publicidade

Scareware -> realizar determinadas ações com base no medo

Pishing -> Fraude
